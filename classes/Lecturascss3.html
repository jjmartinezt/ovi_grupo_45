¿QUE ES CC3?

El Nivel 3 de hojas de estilo en cascada (CSS3) es la iteración del estándar CSS utilizado en el diseño y el formato
de páginas web. CSS3 incorpora el estándar CSS2 con algunos cambios y mejoras.

Un cambio clave es la división del estándar en módulos separados, lo que facilita su aprendizaje y comprensión. A partir de febrero de
2014, el World Wide Web Consortium (W3C) aún está desarrollando el estándar, pero varias de las propiedades CSS3 se han implementado en
las últimas versiones de algunos navegadores web.
